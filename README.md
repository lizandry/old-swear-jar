# swear jar

### Setup Instructions

- `createdb swear-jar`
- `psql swear-jar < swear-jar.sql`
- `cd server`
- `npm install`
- `npm start`
- `..`
- `cd client-app`
- `npm install`
- `npm start`
* NOTE: getting react-app started might be weird, its main dependencies are tsc and webpack
